### 第一部分：方案一實作 (前端 JavaScript 自動造句)

請將這段程式碼加入你的 `prompt-builder.html` 的 `setup()` 區塊中。這會利用簡單的邏輯，把你的參數串成一個通順的句子。

**新增 computed 屬性 `narrativeOutput`：**

JavaScript

```
// 在 setup() 內部定義
const narrativeOutput = computed(() => {
  const f = form.value; // 簡寫
  let sentence = "";

  // 1. 核心主體與動作 (Subject + Action)
  // 如果有動作，通常接在主體後；如果沒有，主體就是核心
  if (f.description) {
     // 如果使用者手填了主要描述，直接用它
     sentence = f.description.trim();
  } else if (f.subject) {
     sentence = f.subject;
     if (f.action) sentence += ` ${f.action}`; // 例如: A cat floating in space
  }

  // 確保句子以句號結尾，方便後續串接
  if (sentence && !sentence.endsWith('.')) sentence += ".";

  // 2. 環境 (Environment) - 使用 "Set in/on" 或直接串接
  if (f.environment) {
    // 判斷環境字串是否已經包含介系詞 (in, on, at)
    const hasPreposition = /^(in|on|at|inside|outside|under)\s/i.test(f.environment);
    if (hasPreposition) {
      sentence += ` The scene takes place ${f.environment}.`;
    } else {
      sentence += ` It is set in ${f.environment}.`;
    }
  }

  // 3. 風格與媒材 (Style) - 這是處理「針織/黏土」的關鍵
  if (f.style) {
    // 簡單暴力的萬用句型
    sentence = `A ${f.style} artwork of ${sentence}`;
    // 如果是寫實照片，修正語法 (可選)
    if (f.style.toLowerCase().includes('photo')) {
      sentence = sentence.replace('artwork of', 'photograph of');
    }
  }

  // 4. 光線與氛圍 (Lighting)
  if (f.lighting) {
    sentence += ` The lighting is ${f.lighting}.`;
  }

  // 5. 技術參數 (Camera/Lens)
  const techParams = [];
  if (f.shotSize) techParams.push(f.shotSize);
  if (f.angle) techParams.push(`shot from a ${f.angle}`);
  if (f.cameraMovement) techParams.push(`with ${f.cameraMovement}`);

  if (techParams.length > 0) {
    sentence += ` Captured ${techParams.join(', ')}.`;
  }

  // 6. 細節 (Details)
  if (f.details) {
    sentence += ` Featuring ${f.details}.`;
  }

  return sentence;
});
```

例如：

- **針織** -> LLM 應該聯想到：_woolen texture, soft yarn, knitted patterns, fuzzy_.
- **黏土** -> LLM 應該聯想到：_fingerprints, stop-motion look, plasticine texture, studio lighting_.

1. 語法微調：解決 "A" vs "An" 的問題
   你目前的輸出是：

"A Art Nouveau, Mucha style..."

在英文中，母音 (a, e, i, o, u) 開頭的單字前應該用 An。這是一個小細節，但改掉會讓質感大增。

🔧 優化程式碼 (Helper Function)： 請在 setup() 裡加入這個小函式，並在組合句子時呼叫它。

JavaScript

// 判斷要用 A 還是 An
const getArticle = (word) => {
// 如果單字不存在，預設回傳空字串
if (!word) return '';
// 檢查第一個字母是否為母音 (不分大小寫)
const firstLetter = word.trim().charAt(0).toLowerCase();
return ['a', 'e', 'i', 'o', 'u'].includes(firstLetter) ? 'An' : 'A';
};

// --- 在 computed narrativeOutput 中使用 ---

const stylePrefix = getArticle(f.style);
