Pi → 筆電同步
是的，需要手動同步。mempalace 讀的是本機靜態檔案，不會自動跟 Pi 連線。

每次 Pi 上的 murmur 有大改動後，要重新做一次：

# Pi 上打包
cd ~
tar czf murmur-export.tar.gz --exclude='data/*.db*' --exclude='.venv' --exclude='node_modules' --exclude='.env' murmur/

傳到筆電後覆蓋 murmur-mirror，再重新挖掘：

mempalace mine C:\Users\USER\murmur-mirror

不用頻繁做，大功能完成或想查東西之前同步一次就好。

筆電上的日常使用
Claude Desktop 掛上 MCP 後，不需要任何手動操作。正常對話就好，Claude 會自己判斷何時呼叫 mempalace 搜尋。

你也可以主動要求，例如：

「從 mempalace 搜尋 reminder 怎麼實作的」
「列出所有 wings」
mempalace 本身也可以手動存新知識（跟 murmur 無關的筆記）：

mempalace search "某個關鍵字"

或在 Claude Desktop 對話中直接說「幫我記住 xxx」，Claude 會用 mempalace_add_drawer 存進去。
