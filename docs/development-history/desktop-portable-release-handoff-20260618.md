# Desktop Portable Release Handoff Archive (2026-06-18)

This archive preserves completed desktop-shell and release-governance details that no longer belong in `HANDOFF.md` or the active section of `docs/TODO.md`.

## Final Current Truth

- Go primary is the only product runtime. Python Flask backend source was removed in T053.
- Windows desktop portable is the current public Windows package path.
- `Prism.exe` is a Windows GUI executable linked with `-H=windowsgui` and `main.desktopShellDefault=1`.
- Desktop mode starts the Go primary runtime in the same process, waits for `/healthz`, then opens the local UI in WebView2.
- The default double-click data directory is the executable-neighbor `PrismData\`.
- `--data-dir` and `PRISM_GO_DATA_DIR` remain advanced/debug overrides.
- The runtime does not show a first-run data-dir selector, does not write `PrismPortable.json`, and does not create or repair desktop shortcuts.
- Prompt Builder seed config is packaged under `static/config/prompt_options.json` and `static/config/wizard_options.json`.
- `Prism.ico` is generated by `scripts/generate_prism_icon.ps1`, embedded by `scripts/generate_windows_resource.ps1`, and loaded by the desktop shell for window/tray icon use.
- Windows Server Dashboard hides the CPU temperature card on no-temperature platforms and uses the data-location card in that slot.
- Pi deployment remains the linux/arm64 Go artifact plus `prism-go-primary.service`, Caddy, and `DEPLOY-PI.md`; it does not include WebView2, tray, Windows mutex, Windows GUI flags, installer, or updater logic.

## Language and First-Run Behavior

- Active UI i18n is implemented for `zh-TW`, `en`, `ja`, and `ko`.
- Current first-run behavior reads `localStorage["locale"]` first; if missing, it defaults to English (`en`) without OS/browser language detection.
- Users can manually switch Traditional Chinese (`zh-TW`), English (`en`), Japanese (`ja`), or Korean (`ko`) in Settings > Appearance; the choice persists in `localStorage`.

## Release Documentation Decisions

- GitHub default README is now English in root `README.md`.
- Traditional Chinese README lives in `README.zh-TW.md`.
- Both files have top language links.
- Release asset naming target for this line is `PrismDesktopPortable-v2.4.9.zip`.
- Public release wording must not claim an installer, auto updater, WebView2 bootstrap, built-in auth, semantic search, bundled AI, or Python backend runtime.

## Deferred Gates

Only reopen a desktop packaging decision gate when one of these is explicitly needed:

- Start Menu shortcut
- desktop shortcut
- file association
- custom data-folder picker
- Windows account data folder default
- WebView2 Runtime bootstrap
- uninstall flow
- update flow

Until then, do not introduce NSIS, WiX, MSIX, auto updater, shortcut automation, hidden PowerShell, or installer-specific state.
